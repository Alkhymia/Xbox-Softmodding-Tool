These instructions are for use, only if you used "Factory Reset" option on the "2016 Softmodding Tool Extras Disc".

How to re-enable the softmod.

1. Power on the system.

2. Now select Music
	> The blank soundtrack
	> Select Copy
	> Select Copy
	> Select New Soundtrack
	> Enter <<Eggsßox>> ( with two '<' Symbols, capital 'E', beta Accent 'ß' and two '>' Symbols )
	> Select Done
	> Now the MS logo will appear & the MSdash will reload.
	
3. Now select music
	> Select blank soundtrack
	> Select Copy
	> Select Copy
	> Select New Soundtrack
	> Select Done
	> Resoftmod your Xbox.

Now that's you done :)
