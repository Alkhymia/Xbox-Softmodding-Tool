● create_md5sums – Create MD5 sums

 

        create_md5sum [Options] [Root Directory]

        This tool will produce a file named 'md5sums' used by the md5_checker

        Xbox program (included in the archive). FTP the checker's 'default.xbe'

        along with the 'md5sums' file and run it on your Xbox to check that all files

        on your DVD-R are readable and 100% identical to the files on your hard

        drive.

           

        Options:

            -o [Filename] Will output the md5sums to the specified file.
